Setup
  donate-cli init                    Create agent
  donate-cli status                  Check status

Brokers
  donate-cli connect alpaca          Connect Alpaca
  donate-cli connect coinbase        Connect Coinbase
  donate-cli connect kraken          Connect Kraken
  (Also: robinhood, interactive-brokers)

Rules
  donate-cli rule create -t percentage -v 0.1    Create 0.1% rule
  donate-cli rule list               Show rules
  donate-cli rule pause [id]         Disable
  donate-cli rule delete [id]        Remove

Donations
  donate-cli donate                  Make donation
  donate-cli donations list          View history
  donate-cli donations summary       Show stats

Agents
  donate-cli agent list              List agents
  donate-cli agent create --name bot2
  donate-cli agent deploy [name]     Deploy to cloud

Special
  donate-cli telegram setup          Setup Telegram
  donate-cli openclaw --token TOKEN  Setup cloud
